📊 E-commerce Sales Dashboard (Power BI)
📌 Project Overview

This project is an interactive Power BI dashboard that analyzes e-commerce sales data. It helps visualize business performance, track key metrics, and identify trends to support data-driven decision-making.

🛠️ Tools Used
Power BI Desktop
Power Query (Data Cleaning)
DAX (Data Modeling & Measures)
Excel / CSV Dataset
📈 Key Insights
Total sales and revenue analysis
Monthly and yearly sales trends
Top-selling products and categories
Region-wise performance analysis
Customer purchasing behavior
